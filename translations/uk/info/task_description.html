Це завдання полягає у створенні розв'язувача для головоломки cryptarithmetic puzzle. У цій головоломці кожній букві потрібно поставити у відповідність одноцифрове число, щоб завершити обчислення.

Вам задано список слів. Останнє слово є сумою. Ви повинні повернути словник, у якому алфавіти є ключами, а одноцифрові числа - значеннями.

ЗАУВАЖЕННЯ:

  • Кожна літера повинна відповідати окремій цифрі.
  • Перша цифра багатозначного числа не повинна бути нулем.
  • Всі тести завжди мають одну відповідь.

Приклади: 

assert cryptarithm_solver(['SEND', 'MORE', 'MONEY']) == {'S': 9, 'E': 5, 'N': 6, 'D': 7, 'M': 1, 'O': 0, 'R': 8, 'Y': 2}
assert cryptarithm_solver(['SIX', 'SEVEN', 'SEVEN', 'TWENTY']) == {'Y': 4, 'X': 0, 'N': 2, 'I': 5, 'E': 8, 'T': 1, 'S': 6, 'V': 7, 'W': 3}

Вхідні дані: Список слів (A list of string)

Вихідні дані: Словник, у якому алфавіти відповідають одноцифровим числам (A dictionary)

Передумова:

  • len(words) <= 10
