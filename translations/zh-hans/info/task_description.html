这是一项为cryptarithmetic puzzle 创建解题程序的任务。 在这个谜题中，每个字母都必须分配一个一位数的数字才能完成计算。

您会得到一个单词表。 最后一个单词就是总和。 您必须返回一个以字母为键、以一位数字为值的字典。

注意：

  • 每个字母应代表不同的数字。
  • 多位数的起始位不得为零。
  • 所有测试都只有一个答案。

举例说明 

assert cryptarithm_solver(['SEND', 'MORE', 'MONEY']) == {'S': 9, 'E': 5, 'N': 6, 'D': 7, 'M': 1, 'O': 0, 'R': 8, 'Y': 2}
assert cryptarithm_solver(['SIX', 'SEVEN', 'SEVEN', 'TWENTY']) == {'Y': 4, 'X': 0, 'N': 2, 'I': 5, 'E': 8, 'T': 1, 'S': 6, 'V': 7, 'W': 3}

输入：单词列表（字符串列表）

输出：字母与一位数数字对应的字典（字典）

先决条件

  • len(words) <= 10
