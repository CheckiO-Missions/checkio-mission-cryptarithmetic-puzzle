Dies ist ein Auftrag, einen Solver für das cryptarithmetic puzzle zu erstellen. Bei diesem Rätsel muss jedem Buchstaben eine einstellige Zahl zugeordnet werden, um die Berechnung abzuschließen.

Sie erhalten eine Liste von Wörtern. Das letzte Wort ist die Summe. Sie müssen ein Wörterbuch mit den Alphabeten als Schlüssel und den einstelligen Zahlen als Werte zurückgeben.

HINWEIS:

  • Jeder Buchstabe sollte für eine andere Ziffer stehen.
  • Die führende Ziffer einer mehrstelligen Zahl darf nicht Null sein.
  • Alle Tests haben immer eine Antwort.

Beispiele: 

assert cryptarithm_solver(['SEND', 'MORE', 'MONEY']) == {'S': 9, 'E': 5, 'N': 6, 'D': 7, 'M': 1, 'O': 0, 'R': 8, 'Y': 2}
assert cryptarithm_solver(['SIX', 'SEVEN', 'SEVEN', 'TWENTY']) == {'Y': 4, 'X': 0, 'N': 2, 'I': 5, 'E': 8, 'T': 1, 'S': 6, 'V': 7, 'W': 3}

Eingabe: Die Liste der Wörter (Eine Liste von Zeichenketten)

Ausgabe: Das Wörterbuch, in dem die Alphabete den einstelligen Zahlen entsprechen (Ein Wörterbuch)

Vorbedingung:

  • len(Wörter) <= 10
