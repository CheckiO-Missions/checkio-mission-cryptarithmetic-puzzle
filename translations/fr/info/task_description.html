Il s'agit d'une mission visant à créer un solveur pour le cryptarithmetic puzzle. Dans ce puzzle, chaque lettre doit être associée à un chiffre pour que le calcul soit complet.

Vous disposez d'une liste de mots. Le dernier mot est la somme. Vous devez retourner un dictionnaire avec les alphabets comme clés et les nombres à un chiffre comme valeurs.

REMARQUE:

  • Chaque lettre doit représenter un chiffre différent.
  • Le premier chiffre d'un nombre à plusieurs chiffres ne doit pas être nul.
  • Tous les tests ont toujours une seule réponse.

Exemples: 

assert cryptarithm_solver(['SEND', 'MORE', 'MONEY']) == {'S': 9, 'E': 5, 'N': 6, 'D': 7, 'M': 1, 'O': 0, 'R': 8, 'Y': 2}
assert cryptarithm_solver(['SIX', 'SEVEN', 'SEVEN', 'TWENTY']) == {'Y': 4, 'X': 0, 'N': 2, 'I': 5, 'E': 8, 'T': 1, 'S': 6, 'V': 7, 'W': 3}

Entrée: La liste des mots (Une liste de chaînes de caractères)

Sortie: Le dictionnaire des alphabets correspondant aux nombres à un chiffre (Un dictionnaire)

Condition préalable:

  • len(words) <= 10
