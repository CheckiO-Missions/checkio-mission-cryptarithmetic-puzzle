Jest to misja polegająca na stworzeniu rozwiązania dla strony cryptarithmetic puzzle. W tej łamigłówce każda litera musi mieć przypisaną jednocyfrową liczbę, aby zakończyć obliczenia.

Otrzymują Państwo listę słów. Ostatnie słowo jest sumą. Proszę zwrócić słownik z alfabetami jako kluczami i jednocyfrowymi liczbami jako wartościami.

PROSZĘ ZAUWAŻYĆ:

  • Każda litera powinna reprezentować inną cyfrę.
  • Pierwsza cyfra liczby wielocyfrowej nie może być zerem.
  • Wszystkie testy mają zawsze jedną odpowiedź.

Przykłady: 

assert cryptarithm_solver(['SEND', 'MORE', 'MONEY']) == {'S': 9, 'E': 5, 'N': 6, 'D': 7, 'M': 1, 'O': 0, 'R': 8, 'Y': 2}
assert cryptarithm_solver(['SIX', 'SEVEN', 'SEVEN', 'TWENTY']) == {'Y': 4, 'X': 0, 'N': 2, 'I': 5, 'E': 8, 'T': 1, 'S': 6, 'V': 7, 'W': 3}

Wejście: Lista słów (lista ciągów znaków)

Dane wyjściowe: Słownik, w którym alfabetom odpowiadają jednocyfrowe liczby (A dictionary)

Warunek wstępny:

  • len(words) <= 10
